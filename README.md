# Vincent Tailoring - Customer Tracker PWA

A secure, mobile-friendly web application for managing customer details, measurements, and photos.

## Features
- **Customer Database**: Store Name, Phone, and Book ID in Google Sheets.
- **Photo Management**: Upload and view measurement photos directly to Google Drive.
- **Search**: Fast search by Name, Phone, or ID.
- **Mobile First**: Designed as a PWA to look and feel like a native app on your phone.

## Setup
1. This app uses Google Identity Services.
2. Requires a Google Cloud Project with Sheets and Drive API enabled.
3. Configure `CLIENT_ID` in `app.js`.
